# U.S. Oil Production Dashboard

Interactive dashboard showing U.S. oil production by state with profitability analysis.

## Features
- State-by-state production data
- Profitability analysis with current oil prices
- Historical trends (2019-2025)
- Oil uses breakdown
- Top operating companies by region

## Installation
```bash
npm install
npm run dev
```

## Data Sources
- U.S. Energy Information Administration (EIA)
- Bloomberg
- Rystad Energy
